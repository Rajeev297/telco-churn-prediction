🐛 TROUBLESHOOTING

Issue: "Module not found" error
  Solution: pip install -r requirements.txt

Issue: XGBoost won't install
  Solution: pip install --upgrade setuptools wheel
           pip install xgboost

Issue: Streamlit port in use
  Solution: streamlit run app/streamlit_app.py --server.port 8502

Issue: Out of memory
  Solution: In train_pipeline.py, reduce sample size:
           self.df_raw = self.df_raw.sample(n=5000)

For more help: See README.md Troubleshooting section
